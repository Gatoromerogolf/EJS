Explicación: Estructura HTML: Un contenedor principal (.chart-container) para agrupar los gráficos. Cada gráfico está dentro de un div con la clase .chart-item. Cada canvas para el gráfico tiene un id único. CSS: .chart-container: Define un contenedor flexible que envuelve todos los gráficos. .chart-item: Establece el ancho y el margen para que los gráficos se dispongan adecuadamente. JavaScript: Utiliza Chart.js para inicializar cada tipo de gráfico. Cada gráfico se inicializa con datos y opciones de ejemplo. Este archivo HTML incluirá y mostrará todos los tipos de gráficos mencionados en la página cuando se abra en un navegador. Puedes ajustar los datos y las opciones según tus necesidades específicas.
